Quick start

To start up Savory.js you need to include savory.min.js script to your pages and add property "id" with value "container" to your main container (the node that is between a header and a footer). In case you do not have this container - it is necessary to create it and add id="container" to it.

Default config will be used after a page loading. You can see its options here

In order to Savory started working out, in the simplest case the page should look like this:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Simple site</title>
    </head>
    <body>
        <header>
            Your header and menu is here
        </header>
        <div id="container">
            This is main container. Loaded content will be appeared here. 
        </div>
        <script src="path/to/savory.min.js"></script>
    </body>
</html>

You can learn more on get started page
