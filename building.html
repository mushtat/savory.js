Build Savory locally

For building Savory.js from its sources used node.js, node package manager (npm) and Browserify. You need to install node to continue.

Download the files of the Savory archive from github repository and unpack it in a different folder or clone repository.

In the src folder run "build.bat" on Windows or "build.sh" on GNU/Linux. This shell script will build the minified file "savory.min.js".
