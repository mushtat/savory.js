Audio player with no page reloading

Savory.js change static site to AJAX site. If this static site have audio player in non-reloadable place - this played will play thru page changing.

This site is live example of such realization. But you can download light version with only audio player.

Important: - this example (and all Savory.js examples) need web-server to work properly.

This can be done simply with PHP or Python, for example. If you want to use this tools for local web-server, you need to download and install Python or PHP and execute following code in command line in your project folder:

Python 2.* example:

python -m http.server 8000

Python 3 example:

python -m SimpleHTTPServer 8000

PHP >5.2 example:

php -S localhost:8000
